This file contains some main code developed under the program maths4life 2021 at the Barcelona's institute for research in biomedicine (IRB).

The results and the full study is stated in the document Maths4Life_report_TeodorParella.pdf.

An example of autosklearn automatic machine learning optimizing task is included in AutoSKlearnTask file.

Classifier_test file contains the test of the previously optimized autosklearn. It also includes calculation of corresponding precision-recall and roc curves.

Signaturizer_applicability file, contains a study of the different confidence values (applicability) for all the spaces in the Chemical checker.
